  这是ddp训练吗                                                                                                                                                                                                              
                                                                                                                                                                                                                             
  是的，是 DDP（Distributed Data Parallel）训练。                                                                                                                                                                            
                                                                                                                                                                                                                             
  具体来说用的是 HuggingFace Accelerate 框架封装的 DDP，底层通信是 PyTorch DDP + NCCL。                                                                                                                                      
                                                                                                                                                                                                                             
  - 16 个进程（每 GPU 一个），每个进程处理不同的数据 batch                                                                                                                                                                   
  - 每个 step 结束后通过 NCCL all_reduce 同步梯度                                                                                                                                                                            
  - 节点内走 NVLink，节点间走 EFA RDMA                                                                                                                                                                                       
  - 有效 batch size = 16卡 × 2 × 4（grad_accum）= 128                                                                                                                                                                        
                                                                                                                                                                                                                             
  不是 FSDP（模型分片），也不是 Tensor Parallel，模型在每张卡上都有完整副本。
